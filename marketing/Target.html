System Manual

B2B 타겟 마케팅 & 자동 온보딩 시스템

구축 완료된 분류 접근 제어 로직 및 마케팅 운영 표준 가이드입니다.

1. 분류 접근 제어 기능 명세 (System Spec)

현재 시스템에 적용된 카테고리 보안 및 노출 로직 정의입니다.

🛠️ 관리자 설정 값 (Configuration)

설정 위치 쇼핑몰관리 > 분류관리 > 분류 수정
핵심 필드 지정 유통사 코드 (Target Code)
ℹ️ 기능 정의: 이 필드에 코드(예: WH_A)를 입력하면, 해당 카테고리는 '전용 분류'로 전환되어 일반 접근이 차단됩니다.

⚖️ 접근 제어 로직 (Access Logic)

🔒 설정 방법

관리자 페이지의 [분류 수정] 메뉴에서 지정 유통사 코드 필드에 코드(예: WH_A)를 입력하면, 해당 카테고리는 즉시 외부 접근이 차단된 '전용 분류'가 됩니다.

⛔ 차단 대상

코드가 없는 일반 회원/비회원과 코드가 다른 타 유통사 회원은 절대 접근할 수 없습니다.

✅ 허용 대상

설정된 코드와 정확히 일치하는 '자사 전용회원'(로그인) 및 '마케팅 링크 접속자'만 입장할 수 있습니다.

2. 마케팅 운영 플로우 (Operation Flow)

시스템 기능을 활용하여 실제 마케팅을 집행하는 표준 절차입니다.

1

상품 및 분류 세팅

  • 상품 등록: 마케팅용 전용 상품 등록
  • 분류 설정: 해당 상품을 '전용 카테고리(예: 도매몰)'로 이동
  • 코드 입력: 분류 수정 화면에서 지정 유통사 코드(yt_code) 입력 (잠금 설정)
2

랜딩 페이지 제작 & 링크 삽입

[작업] 구매 버튼(CTA) 링크 설정

<a href="/shop/list.php?ca_id=10&yt_code=WH_A"> 입장하기 </a>

※ 이 '마케팅 접속 링크'가 고객에게 임시 권한을 부여합니다.

3

고객 유입 및 자동화 (System Action)

  • ✔ 접속: 링크 클릭 시 비로그인 상태로 전용 상품 열람 가능 (임시 권한)
  • ✔ 가입: 회원가입 완료 시 '전용회원(WH_A 소속)'으로 DB 자동 저장
