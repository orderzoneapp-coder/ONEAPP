Project Briefing

B2B 폐쇄몰 타겟 마케팅 및
자동 온보딩(Auto-Onboarding) 시스템

"마케팅 링크 하나로 [비로그인 열람 + 회원가입 + 등급상향 + 거래처매칭]을
원스톱으로 처리하는 지능형 유입 프로세스"

1. 기획 목적 및 기대 효과

🔓

진입 장벽 제거 (선 노출)

폐쇄몰의 단점인 '가입 전 상품 확인 불가' 문제를 해결합니다. 링크 보유자에게는 로그인 없이도 도매가를 먼저 보여주어 가입 유도율을 극대화합니다.

⚡

가입 프로세스 자동화

"가입 → 승인대기 → 승인 → 등급조정"의 복잡한 절차를 없앱니다. 가입 즉시 'Lv.5 유통사' 등급과 '전용회원' 자격을 부여합니다.

🛡️

일반 고객 완벽 차단

링크가 없는 일반 소매 고객(Lv.2)에게는 도매 카테고리와 상품이 검색조차 되지 않도록 철저히 격리합니다.

2. 핵심 프로세스 (Workflow)

1

접속 (Link)

마케팅 이메일 클릭
?key=PROMO_A

[System Logic] 권한 부여
  • ✅ URL 키 검증 및 세션(Session) 생성
  • 🎁 비로그인 상태에 '임시 Lv.5' 권한 부여
  • 👁️ 숨겨진 '도매몰' 카테고리 및 상품 노출 시작
2

탐색 & 가입

상품 확인 후
회원가입 진행

[System Logic] 자동 매칭
  • 💾 가입 시 세션에 저장된 '유통사 코드(WH_A)' 로드
  • 🆙 회원 등급을 'Lv.5 (유통사)'로 강제 상향 저장
  • 🔗 전용 유통사 코드를 'WH_A'로 자동 입력
3

완료

즉시 거래 가능

[Final State]

고객은 별도 승인 없이 즉시 도매가 구매가 가능하며,
관리자는 별도의 매칭 작업 없이 매출을 발생시킬 수 있습니다.

3. 대상별 노출 로직 (Access Matrix)

접속 대상 접속 경로 권한 상태 도매몰 노출 비고
일반 방문자 일반 URL Lv.0 (비회원) ⛔ 차단 카테고리 숨김, 검색 불가
일반 소매 회원 로그인 Lv.2 (일반) ⛔ 차단 레벨 제한에 걸림
마케팅 타겟 마케팅 링크 가상 Lv.5 ✅ 노출 세션 권한으로 통과
유통사 (전용회원) 로그인 Lv.5 (유통사) ✅ 노출 정상 접근
